# AR Replay Signal Testbed — Curation Rubric

_Generated by `scripts/select_testbed.py`. Pool: top **80** trials from `attentional-foraging/AdSERP/data/trial-scores.csv` after per-participant diversity capping. Final testbed: **28** trials across 6 groups._

## What this is

This is a **signal testbed**, not a showcase. Each trial is a tagged demonstration of the four-class taxonomy inference task: SERP screenshot + cursor + per-AOI labels overlaid. The viewer renders what the AR library's `Episode` classifier would emit on this cursor stream.

Per-AOI labels are **predictions**, not ground truth — derived from cursor enter/dwell/exit episodes against AOI bboxes via `derive_aoi_labels()` in `scripts/build_replay_trial.py`. If you disagree with a label on a specific trial, that trial belongs in Group F (failure modes) and the inference logic needs work.

## Inference logic (current)

For each AOI bbox, walk the cursor stream. An **episode** is an enter→exit traversal where dwell ≥ 100ms (filters drive-by crossings). Label rules:

- `CLICKED` — any episode contains a click event inside the bbox.
- `DEFERRED` — ≥2 kept episodes (cursor returned after leaving).
- `EVALUATED_REJECTED` — exactly 1 kept episode (entered, left, never returned).
- `NOT_APPROACHED` — 0 kept episodes (cursor never entered, or only crossed briefly).

## Selection rubric

1. **Pass 1** (`scripts/score_trials.py` in attentional-foraging) scores every trial on cursor-only features:
   ```
   score = 0.20 * normalize(n_mousemove,    50, 400)
         + 0.25 * normalize(distinct_y_bands, 2,   8)
         + 0.25 * normalize(y_band_revisits,  0,   6)
         + 0.15 * normalize(duration_ms,   5000, 25000)
         + 0.15 * has_click
         × (1 - 0.5 * pupil_dropout_rate)
   ```
   `has_click` is 1.0 for every trial in AdSERP (protocol-required) — kept for transparency but doesn't discriminate.

2. **Pass 2** (this script) takes the top **80** with a per-participant cap, extracts organic AOI bboxes, builds replay bundles with derived labels, and assigns each trial to the most diagnostic group it qualifies for (priority order: E → B-DEF → B-REJ → C → D → A).
