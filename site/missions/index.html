Movie missions

Pick a movie. We'll write your prompt.

Six brief missions — date night, kids, extended family, solo wind-down, teen sleepover, rainy Sunday afternoon. For each, you'll see ten classics and pick the one you'd actually watch. We turn your picks into a copy-pasteable recommendation prompt for ChatGPT, Claude, or Poe.

What we record

Which movie you picked, which ones you considered (you looked, you thought about it), and which ones you rejected (you looked and turned away). The prompt encodes all three so the LLM has more than a thumbs-up list to work with.

All approach/retreat layouts

Cursor data is captured anonymously via PostHog (or not at all in private browsing). You can clear your run state any time from the prompt page.
